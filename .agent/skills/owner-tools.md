# 오순이 스킬 정의 파일 (owner-tools.md)

> 이 파일은 오순이 챗봇의 확장 스킬을 정의하는 공간입니다.
> 향후 비즈니스 요구에 따라 스킬을 추가하면 챗봇이 자동으로 학습합니다.

## 현재 활성 스킬

*(아직 정의된 스킬이 없습니다. 필요에 따라 아래 형식으로 추가하세요.)*

## 스킬 추가 형식

```
### [스킬명]
- **트리거**: 어떤 상황에서 이 스킬을 사용하는가
- **동작**: 챗봇이 수행해야 할 행동
- **예시 질문**: 사용자가 할 수 있는 질문 예시
```

## 스킬 추가 예정 목록

- [ ] 영업 현황 요약 리포트
- [ ] 계약 진행 상태 안내
- [ ] 교육 일정 자동 추천
- [ ] 리더님별 맞춤 정보 제공

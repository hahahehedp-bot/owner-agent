# 🛠️ 오순이 전용 기술 (Owner Skills)

오순이는 다음 기술들을 사용하여 리더님들께 정확한 정보를 제공합니다.

## 1. 일정 조회 기술 (`get_schedule`)
- **목적**: `data/schedule.csv` 파일을 읽어 최신 오너 세미나 및 교육 일정을 안내합니다.
- **활용 상황**: 
  - "이번 주 일정이 어떻게 돼?"
  - "유여름 파트너님 교육 언제야?"
  - "오늘 강연 장소가 어디지?"

## 2. 자료실 검색 기술 (`get_resources`)
- **목적**: `data/resources.csv` 파일을 읽어 제품 설명서, 영상, PPT 자료 링크를 제공합니다.
- **활용 상황**:
  - "리버스12 카탈로그 보내줘."
  - "세노이드 홍보 영상 어디 있어?"
  - "PPT 자료 다운로드하고 싶어."

## 3. 마케팅 프롬프트 생성 기술 (`marketing_booster`)
- **목적**: 리더님의 요청에 따라 SNS, 블로그 등에 최적화된 마케팅 문구를 실시간 생성합니다.
- **활용 상황**:
  - "인스타에 올릴 짧은 문구 짜줘."
  - "고객한테 보낼 카톡 정중하게 써줘."

---
*모든 기술은 구글 앱스 스크립트(GAS) 백엔드와 연동되어 실시간으로 작동합니다.*
